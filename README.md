# basketball
Le basketball est un sport collectif inventé en 1891 par James Naismith aux États-Unis. Il oppose deux équipes cherchant à marquer des points en lançant un ballon dans un panier placé à 3,05 m du sol. Le jeu demande vitesse, coordination et stratégie. Aujourd’hui, il est pratiqué dans le monde entier et très populaire grâce à la NBA.
